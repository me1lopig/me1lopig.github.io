# Bienvenidos a mi nueva web

¡Hola a todos! Acabo de lanzar mi nueva página web construida con **Jekyll** y alojada en **GitHub Pages**. Aquí podréis encontrar:

- **Todos mis repositorios de GitHub** en un solo lugar.
- **Posts técnicos** sobre proyectos, tutoriales y más.

## ¿Por qué Jekyll?

Elegí Jekyll porque:

1. **Es simple**: No necesito base de datos ni backend.
2. **Es rápido**: Los sitios estáticos cargan en milisegundos.
3. **Es gratis**: GitHub Pages lo soporta nativamente.
4. **Es personalizable**: Puedo adaptarlo a mis necesidades.

## ¿Qué podéis esperar?

- **Actualizaciones frecuentes** de mis repositorios.
- **Posts técnicos** sobre desarrollo, mecánica y más.
- **Tutoriales** para ayudar a otros desarrolladores.
